Youngest Modern Farmer - Multi-page HTML site
Files:
- index.html, about.html, programs.html, blog.html, partners.html, contact.html
- css/style.css
- images/ (logo.png, hero.svg, program.svg, blog1.svg, blog2.svg, blog3.svg)

How to test:
1. Unzip the package and open index.html locally in your browser.
2. To deploy on GitHub Pages:
   - Create a new repository, push the site files to the repo (root or docs/), then enable GitHub Pages in repo settings.
